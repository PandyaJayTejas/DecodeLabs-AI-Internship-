## 🗂️ Projects

### 🤖 Project 1: NEXUS - Rule-Based AI Chatbot

A lightweight conversational chatbot that responds using predefined rules and pattern matching. The system focuses on speed, reliability, and predictable responses without relying on external AI models.

**Key Skills Demonstrated:**

* Dictionary-Based Knowledge Systems
* Input Sanitization
* Rule-Based Decision Making
* Command-Line Interface Design

**Run:**

```bash
python Project2.py
```

### 📊 Project 2: Sentiment Analyzer

A text analysis tool that determines whether input text expresses positive, negative, or neutral sentiment. The analyzer uses weighted keyword scoring and negation detection to improve accuracy.

**Key Skills Demonstrated:**

* Text Processing
* Tokenization
* Sentiment Scoring
* Regular Expressions
* Data Analysis Logic

**Run:**

```bash
python Project1.py
```
